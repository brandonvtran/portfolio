Dog or Nah

Javascript, CSS, and HTML based web-application that takes pop culture and turns it into a fun and addictive game. The game features the popular dog and inanimate object meme as well as a one second time constraint to make the game challenging.

As a group, we brainstormed for a fun and interesting idea for a project that would serve the purpose of entertaining the masses. We came up with an idea for a web application game that took our favorite meme and turned it into an addictive and competitive game. In order to elaborate on the web application's purpose in the real world, and plan how the game works, we created a storyboard.

After planning the project, we implemented the functionality of the web application by utilizing Javascript, CSS, and HTML. The game was designed to be easy to understand with intuitive controls. When the game was finally finished, we utilized social media to let users test out the game and give feedback. Upon receiving feedback, we modified the web application with incremental changes to improve it.

ReMemo

A Javascript, HTML, CSS based mobile and web application designed to help people remember their tasks for the day and improve their memory

As a group, we brainstormed for an idea for a project that would serve the purpose of helping folks above the age of forty. We came up with an idea for a mobile and web application that helped the user remember tasks for the day and improve memory for the long term. In order to elaborate on the application's purpose in the real world, and plan how it worked, we created a storyboard.

After planning the project, we created a prototype. After testing the protype, we implemented the functionality of the web application by utilizing Javascript, CSS, and HTML. The application was designed to be easy to be simple yet functional. When the application was finished, we performed usability tests on peers. Upon receiving data on design flaws, we redesigned specific aspects of the application.

Ripple

An android and web application that streamlines a music queuing process. The "DJ" receives song requests, on his or her android application, from the "collaborator" who makes the song requests on his or her mobile web application. It is then the DJ's responsibility to either accept or decline the request such that the song will either be added or not added to the queue to be played. Works on a team of ten and acted as the Senior Systems Analyst. Utilizes the languages of Java, XML, Javascript, HTML, and CSS as well as the Firebase database. The process of building the application was done through the software engineering methodology: agile. Artifacts such as use cases, user story, requirements, test cases, acceptance tests, design use cases, and use case diagrams were created in order to plan and document the process of building the application.

Alternative Breaks

Alternative Breaks is an organization that tackles a variety of social justice issues in communities across the world. This particular service trip is called AB Safe Haven which takes place in Cartago, Costa Rica. AB Safe Haven is partnering with a non-profit organization whose mission is to foster youth and community development through educational material. AB Safe Haven will work with a local orphanage that provides a safe environment for neglected youth and children of impoverished, working, single mothers who need support. The mission is to provide stimulating lesson plans and interactive activities to forge meaningful relationships and encourage self-development.
